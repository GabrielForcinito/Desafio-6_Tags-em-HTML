# 🏷️ Desafio Tags HTML

## 📌 Sobre o projeto

Projeto desenvolvido em HTML com o objetivo de praticar e demonstrar o uso das principais tags HTML, explorando sua aplicação e significado semântico.

## 🛠️ Tecnologias utilizadas

* HTML5

## 🎯 Funcionalidades

* Demonstração de diferentes tags HTML
* Estruturação de conteúdo semântico
* Organização de informações em página única
